CSS Snippet Cheatsheet

Here is some helpful CSS syntax!

Flex-Flow

"flex-flow" is a shortcut for both flex-direction and flex-wrap!


.row-wrap {
    display: flex;
    flex-flow: row wrap;
}

Flexbox Column

Use this to create a Flexbox column layout


    .column {
        display: flex;
        flex-direction: column;
    }

CSS Grid Layout

Build a 12-column layout using CSS Grid


    .grid {
        display: grid;
        width: 100%;
        grid-template-columns:
    repeat(12, 1fr):
    }

Rounded corners

Round box corners with "border-radius"


    .round {
        border-radius: 5%
    }

Hover Visibility

Have items become visible on mouse hover


    .hidden "visible"{
        display: none;
    }
                    
    .hidden:hover "visible"{
        display: block;
    }

Overlay Card with Title

Use position properties and negative margins to raise elements higher than their natural starting point


    .card-header {
        position: relative;
        margin-top: -20px;
    }
